:mod:`convertmodis_gdal` module
------------------------------------------------------------------------
Using GDAL Python bindings it is possible to convert MODIS HDF file
without use MRT. It can create GeoTiff file (or other GDAL supported
formats) or HDF mosaic file for several tiles.
